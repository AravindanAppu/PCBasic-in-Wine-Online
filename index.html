GW-BASIC Emulator (via DOSBox-X)

Instructions:

  1. Click "Load .ZIP" and select your ZIP file containing GW-BASIC.EXE and *.BAS files.
  2. Click "Run GW-BASIC" to start the emulator and auto-execute GWBASIC.EXE.
  3. Your .BAS programs will run. Any LPRINT command outputs to LPT1:, which we redirect to a downloadable text file.
  4. The LPRINT output will be saved as lprint_output.txt and offered for download.

Note: This uses DOSBox-X compiled with WebAssembly. Full compatibility depends on the original program's requirements.
